# CLAUDE.md — 토큰 절약 규칙

## Rule 1 — 읽은 파일은 다시 읽지 않는다
skill 또는 memory에 이미 있는 내용이면 파일 Read 호출을 건너뜁니다.

## Rule 2 — 투기성 도구 호출 금지
"있을 수도 있으니" 형태의 추측성 파일 탐색, bash 실행은 즉시 차단합니다.

## Rule 3 — 병렬 실행 가능한 호출은 동시에 실행한다
순서 의존성이 없는 Read/Grep/Glob 호출은 병렬로 묶어서 실행합니다.

## Rule 4 — 20줄 초과 출력은 서브에이전트로 격리한다
메인 컨텍스트에 20줄 초과 출력이 발생할 작업은 서브에이전트에 위임하고 요약만 반환합니다.

## Rule 5 — 유저가 말한 내용을 반복하지 않는다
"네, 말씀하신 대로~" 식의 앵무새 반복 없이 바로 실행하거나 바로 답변합니다.
